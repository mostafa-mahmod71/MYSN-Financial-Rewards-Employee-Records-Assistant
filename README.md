# MYSN - Financial Rewards & Employee Records Assistant 🚀

**MYSN** هو تطبيق إداري مخصص لتنظيم وحساب المكافآت المالية والمستحقات الشهرية للموظفين بشكل دقيق وسريع. يعمل النظام كأداة تكميلية للأنظمة الكبرى لتسهيل العمليات الحسابية المعقدة واستخراج التقارير الورقية المنظمة.

## ✨ وظائف البرنامج (Key Functions)
- **حساب المكافآت الذكي:** حساب فوري للضرائب والتأمينات (حصة الموظف والشركة) لحظة إدخال القيمة.
- **تحديث مجمعات الضرائب:** تتبع آلي للوعاء الضريبي والتأميني (Boxes) الخاص بكل موظف وتحديثه بعد كل عملية صرف.
- **أرشيف الاستمارات:** البحث في الاستمارات السابقة بناءً على التاريخ أو النوع لاستعراض تفاصيلها.
- **دعم الطباعة:** تجهيز البيانات للعرض والطباعة الورقية (كشوف صرف، إذن قيد) لتسهيل الدورة المستندية.
- **سجل الموظفين:** عرض بيانات الموظف المالية والإدارية الأساسية من قاعدة البيانات.

## 🛠️ التكنولوجيا (Tech Stack)
- **Angular 18:** استخدام Signals و Reactive Forms لإدارة الجداول والبيانات.
- **RxJS:** معالجة تحديثات الموظفين الجماعية (Bulk Updates).
- **Tailwind CSS:** واجهة عصرية تدعم اللغة العربية (RTL).

## 🚀 الهدف من المشروع
توفير حل سريع ودقيق لموظفي الحسابات والإدارة لعمل استمارات الصرف اليدوية أو المخصصة، مع ضمان توافق الحسابات مع القواعد المالية المعمول بها، وتحويل السجلات الورقية إلى سجلات رقمية سهلة البحث.

النسخة الحالية المرفوعة أونلاين تعمل بوضع المحاكاة (Static JSON Mode) لسهولة العرض السريع والمجاني، بينما يحتوي الـ Source Code على الخدمات (Services) والـ Routing المهيأين تماماً للربط المباشر مع API وسيرفر حقيقي في أي وقت.
